Traversal Methods Reference
Method Description Documentation
.children() Returns the child elements of the selected element(s). Can add a selector as an optional argument. .children()
.find() Returns the descendants of each of the selected element(s). Accepts selectors, jQuery objects, or elements as arguments. .find()
.next() Returns the next sibling of each of the selected element(s). Can add a selector as an optional argument. .next()
.prev() Returns the previous sibling of each of the selected element(s). Can add a selector as an optional argument. .prev()
.parent() Returns the parent of each of the selected element(s). Can add a selector as an optional argument. .parent()
.has() Selects descendants of the selected element(s) that match the selector or element. .has()
.is() Returns true if the selected elements match a selector, element, or jQuery object. .is()
.not() Removes the selected element(s) from the set of matched elements. .not()
.first() Selects the first of the set of matched elements. .first()
.last() Selects the last of the set of matched elements. .last()
.contents() Selects the children of the set of matched elements, including text and comments. .contents()
.add() Creates a new jQuery object with the set of matched elements and the selected elements. Accepts a jQuery selector, DOM elements, or HTML snippets. .contents()
.end() Ends the most recent filtering methods and returns the selected elements to their previous state. .end()
.each() Each takes a function as an argument and and iterates over the set of matching elements, invoking the function and applying it to each element in the matching set. .each()
