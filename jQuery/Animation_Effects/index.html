Effects

JQuery includes a set effects with standard animations like hide, fade, and slide. It also simplifies the creation of custom animation as well.

Effect Methods Reference
Method Description Documentation
.hide() Hides the set of selected elements. Similar to setting the element's display property to none. .hide()
.show() Shows the set of selected elements. .show()
.toggle() Toggles the set of selected elements to be displayed or hidden. .toggle()
.fadeIn() Fades in the set of selected elements. .fadeIn()
.fadeOut() Fades out the set of selected elements. .fadeOut()
.fadeToggle() Toggles the set of selected elements to fade in or out. .fadeOut()
.fadeTo() Fades the set of selecting elements to the specified opacity. Requires a specific duration. .fadeTo()
.slideUp() Applies a sliding motion to the set of selected elements. .slideDown()
.slideDown() Applies a sliding motion to the set of selected elements. .slideUp()
.slideToggle() Applies a sliding motion to show or hide the set of selected elements. .slideToggle()
